# MCBE Tweaks

https://github.com/Furzide/MCBE-Tweaks

Guide covering performance and quality-of-life tweaks for Bedrock Edition clients (relevant to players connecting via Geyser). Topics: disabling V-Sync, a Bedrock version switcher, in-game settings recommendations, older/alternative Bedrock builds with better performance, HUD/UI resource packs, a Java-style animations pack, a fog-removal shader, and disabling RTX on Nvidia GPUs.
